## PANCONGBELOKAN
This application is designed to help **Pancong Belokan** manage their inventory more efficiently by replacing manual stock tracking with a digital, easy-to-use system. It allows users to record stock in and out, monitor real-time inventory levels, view usage trends through a simple dashboard, and receive alerts when ingredients or materials are running low. The goal is to give the owner a faster, more accurate way to manage their daily operations so they can avoid shortages, reduce waste, and make better decisions for their business.

## Tech Stack
- Vite
- React
- TypeScript
- shadcn-ui
- Tailwind CSS


### 1. Clone the repository
```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
```

### 2. Install dependencies
```sh
npm install
```

### 3. Start the development server
```sh
npm run dev
```

## Build for production
```sh
npm run build
```

## Preview production build
```sh
npm run preview
```
