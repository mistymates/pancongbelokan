## Tech Stack
- Vite
- React
- TypeScript
- shadcn-ui
- Tailwind CSS

## Getting Started

### 1. Clone the repository
```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
```

### 2. Install dependencies
```sh
npm install
```

### 3. Start the development server
```sh
npm run dev
```

## Build for production
```sh
npm run build
```

## Preview production build
```sh
npm run preview
```
